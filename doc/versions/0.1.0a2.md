# Versão 0.1.0a2

## Desenvolvimento

**$** Nunca utilizar
```bash
python libtest/cli.py
```

**$** Utilizar
```bash
python -m libtest.cli
```

**$** Instalando pytest à parte
```bash
uv pip install pytest
```

**$** Executando
```bash
pytest
```

## Para o comando funcionar fora do `env`

**$** Executando
```bash
uv pip install pipx
pipx install .
```

Com isso o comando funciona de qualquer lugar e sem ativar o `env`.
