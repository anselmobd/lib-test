## Desenvolvimento

Utilizando `uv`.

**$** (Comando executado)
```bash
uv python list
```
**>** Resultado (output) do comando:
```text
...
cpython-3.11.11-linux-x86_64-gnu
...
```

**$**
```bash
uv venv --python 3.11.11
```
**>**
```text
Using CPython 3.11.11
Creating virtual environment at: .venv
Activate with: source .venv/bin/activate
```

**$**
```bash
uv python pin 3.11.11
```

### Depois de criar exemplo básico de programa CLI

**$** (Comandos testados)
```bash
## instala pacores requeridos
uv pip install -r requirements.txt

## instala o programa libtest
uv pip install .

## instala o programa libtest em modo de edição
uv pip install -e .

## instala o programa libtest em ambiente de desenvolvimento, com pacotes que são apenas para `dev`
uv pip install .[dev]

## desinstala o programa libtest
uv pip uninstall .

## desinstala todos os pacotes instalados
uv pip freeze | xargs uv pip uninstall
```

**$** Depois de um `install .` o comando está disponível dentro do `env`
```bash
libtest ola Oxigenai
```

